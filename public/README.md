# KMAX Cinema Booking App

A modern, cinematic ticket booking application featuring KMAX screens and DOLBY APPUS sound systems.

## How to Run
Simply open the `index.html` file in any modern web browser.

## Features
- **Immersive UI**: Dark mode design with neon accents.
- **Interactive Booking**: Visual seat selection with real-time price updates.
- **Responsive Design**: Works on desktop and mobile.
- **Tech Stack**: Vanilla HTML, CSS, and JavaScript (No build steps required).
